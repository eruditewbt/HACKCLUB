# GAMING
Tiny games + solvers (no third-party deps).

## Files
- `halving.py`: Simple halving game skeleton (existing).
- `nim_optimal.py`: Nim with optimal XOR strategy + CLI.
- `tictactoe_minimax.py`: Tic-tac-toe with minimax + alpha-beta + CLI.
- `mcts_tictactoe.py`: Small MCTS agent for tic-tac-toe.

## Run
```bash
python GAMING/tictactoe_minimax.py
python GAMING/nim_optimal.py
python GAMING/mcts_tictactoe.py
```
