# FACTOR_GRAPHS (No Dependencies)

This directory contains a minimal factor graph implementation and belief propagation (sum-product)
for tree-structured graphs.

## Files
- `factor_graph.py`: variables, factors, and a small graph container
- `sum_product.py`: sum-product messages for trees (discrete variables)
- `examples.py`: runnable demos (marginals on small graphs)

## Run
```bash
python -m FACTOR_GRAPHS.examples
```
