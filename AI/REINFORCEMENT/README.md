# REINFORCEMENT (No Dependencies)

This directory contains small reinforcement learning projects using only the Python standard library.

## Projects
- `bandits.py`: epsilon-greedy, UCB1, and Thompson sampling on multi-armed bandits
- `gridworld.py`: simple grid world environment
- `q_learning.py`: Q-learning agent for grid world
- `demo.py`: run a couple demos

## Run
```bash
python -m REINFORCEMENT.demo
```
