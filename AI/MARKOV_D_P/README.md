# MARKOV_D_P
Markov Decision Process (MDP) utilities and classic dynamic programming.

Existing scripts:
- `transportmdp.py`: small MDP environment.
- `valueiteration.py`: value iteration demo (script-style).

Added utilities:
- `mdp_base.py`: minimal MDP interface and helpers.
- `gridworld_mdp.py`: gridworld MDP example.
- `policy_iteration.py`: policy iteration implementation.
- `simulate_policy.py`: rollout simulator for a policy.

## Run
```bash
python MARKOV_D_P/simulate_policy.py
```
